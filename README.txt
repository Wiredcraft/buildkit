# Build Kit

Get started building with Drupal fast.

Build Kit is a basic distro meant to capture elements that are
generally useful and make building Drupal sites and Drupal distros
easier.

## Modules

Build kit includes a series of modules that you will find yourself using in
almost all site builds.

## Themes

Build kit ships with Rubik and Tao. Tao is a base theme which Rubik builds upon.
